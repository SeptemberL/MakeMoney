#个人股票信息、交易系统

施工中

    交易系统愿景
    该交易系统为光影相生自用系统，希望通过此系统来进行股票交易，该系统拥有历史股票数据查询、信号通知、微信通知系统、爬虫、量化等功能。
    信号通知系统希望每天和实时来对加入的股票进行监控，出现各种信号了通过微信通知来通知给我，以此基础进行交易。
    要拥有仓位管理系统，接入AI 对仓位进行分析和管理。
    一个敏锐的市场事件爬虫系统，来时刻监控市场上的一些消息。
    每日固定时间对当日的行情生成一份盘前和盘后报告。
