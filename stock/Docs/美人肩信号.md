## 美人肩（MEIRENJIAN）筛选信号

本信号用于在指定窗口内识别“美人肩”形态，并在筛选器 `/stock_filter` 中作为可勾选条件参与全市场扫描。

## 使用方式（网页筛选器）

- 打开页面：`/stock_filter`
- 勾选：**美人肩形态**
- 可选参数：
  - `minScore`：0~1，命中阈值（默认 0.6）
  - `lookbackBars`：识别窗口长度（默认 120）

页面会把参数打包为 `signalParams`（JSON 字符串）并随 `/api/filter_stocks` 一起发送。

## 使用方式（API）

接口：`GET /api/filter_stocks`

关键参数：

- `filterSignal`：可多次出现。例如 `filterSignal=MEIRENJIAN&filterSignal=DLJG`
- `signalParams`：JSON 字符串，按信号名传参。例如：

```json
{
  "MEIRENJIAN": {
    "minScore": 0.7,
    "lookbackBars": 180
  }
}
```

说明：

- 目前信号实现类：`signals/signal_meirenjian.py`（`Signal_MeirenJian`）
- 输出在内部以 `meirenjian_hit/meirenjian_score/meirenjian_reason/meirenjian_meta_json` 打标（最后一根 K），筛选器会在 `dayRange` 范围内扫描这些标记并输出命中信息。
